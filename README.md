# the_timer

SVG-baserad lektionsklocka i ren HTML/JS/CSS. Inga bibliotek.

## Funktioner

- Lektionsdelar via textarea (en rad = en del)
- Starttid + sluttid eller längd
- Roterad startpunkt baserat på faktisk klocktid
- Donut eller pie-läge (ihålig mitt togglas)
- Drag av segmentgränser (mus + touch)
- Dag-/nattläge med varma pastellfärger
- 2-minuters varning per segment (Web Audio + flash)
- Tidvisare som pil från centrum

## Köra lokalt

Öppna `index.html` i webbläsaren — eller kör `npx serve`.

## Deploy

Kopplad till Vercel via GitHub-repo. Static site, ingen build.
